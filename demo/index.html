todo (invoked without options)

{% todo %} This is line 1 of the test content.

todo id='asdf'

{% todo id='asdf' %} This is line 1 of the test content.
This is line 2. {% endtodo %}

todo span

This is a paragraph with {% todo span %}TODO content{% endtodo %}.

todo alert

{% todo alert %} Test content for alert {% endtodo %}.

todo style='background: #eee; color: green;'

{% todo style='background: #eee; color: green;' %} Test content for style {% endtodo %}.

todo block style='background: #eee; color: green;'

{% todo block style='background: #eee; color: green;' %} Test content {% endtodo %}.

todo class='rounded shadow'

{% todo class='rounded shadow' %} Test content for class {% endtodo %}.

todo mode=develop

todo mode=prod

Nothing should be displayed when in development mode.
Jekyll mode is {{jekyll.environment}}.
